# 01 — VITA CITY 建物デザイン設計書

> **対象**: 建設可能 28 種（B001〜B028）+ 自動生成 2 種（B029・B030）= 合計 30 種
> **目的**: 各軸の建物シルエットと Lv1→Lv5 の成長ロジックを設計し、"同じ家族" で育つ体験を保証する。
> **前提**: `00_DesignOverview.md` の 3 原則（丸く・物語・同じ家族）を全建物で遵守。

---

## 1. 軸ごとのデザイン言語（Design Language）

各軸に固有の**建築要素モチーフ**を決めておく。これが揃うことで「どの軸の建物か」が一目でわかる。

### 1.1 運動軸（緑 #34C759）

| 要素 | 仕様 |
|---|---|
| 主素材 | 明るい木材 + ガラス張り |
| 屋根形 | 片流れ or 大きな庇 |
| 装飾 | 芝生の前庭・小さな旗・ボールやダンベル |
| ライン | **直線的・アクティブ** |
| 窓形 | 大きな横長窓（中が見える = 活動感） |
| キーワード | 開放感・躍動・屋外との境界線を薄く |

### 1.2 食事軸（橙 #FF9500）

| 要素 | 仕様 |
|---|---|
| 主素材 | 煉瓦 + 暖色の漆喰 |
| 屋根形 | 三角屋根 + 煙突 |
| 装飾 | 黒板看板・プランター・野菜・ランプシェード |
| ライン | **曲線的・家庭的** |
| 窓形 | 格子窓 + レースカーテン |
| キーワード | 温もり・手作り・甘い香り |

### 1.3 飲酒軸（紫 #AF52DE）— 自制・代替

| 要素 | 仕様 |
|---|---|
| 主素材 | 石材 + 濃い木 + 障子風パネル |
| 屋根形 | 和風の入母屋 or 平らなモダン屋根 |
| 装飾 | 提灯（青紫）・香炉・水琴窟・小さな枯山水 |
| ライン | **静的・シンメトリー** |
| 窓形 | 円窓・障子・すりガラス |
| キーワード | 静謐・内省・月明かり |

### 1.4 睡眠軸（青 #007AFF）

| 要素 | 仕様 |
|---|---|
| 主素材 | ネイビーの瓦 + 白壁 |
| 屋根形 | 丸いドーム or 尖塔 |
| 装飾 | 星型の窓・三日月看板・暖色ランプ（窓明かり） |
| ライン | **緩やかな曲線** |
| 窓形 | 丸窓・星型の抜き窓 |
| キーワード | 夜・静けさ・守られている感 |

### 1.5 生活習慣軸（ピンク #FF2D55）

| 要素 | 仕様 |
|---|---|
| 主素材 | クリーム色の漆喰 + 淡ピンクの装飾タイル |
| 屋根形 | 寄棟・マンサード |
| 装飾 | 時計・花壇・ベンチ・本のオブジェ |
| ライン | **折衷的・アカデミック** |
| 窓形 | 縦長の上げ下げ窓 |
| キーワード | 温かい公共施設・手を差し伸べる |

### 1.6 市庁舎・総合（金 #FFD700）

| 要素 | 仕様 |
|---|---|
| 主素材 | 白大理石 + 金色のディテール |
| 屋根形 | ドーム + 旗 |
| 装飾 | 時計塔・エンブレム・階段・噴水 |
| ライン | **荘厳だが過剰でない** |
| 窓形 | アーチ型の大窓 |
| キーワード | 街の心臓・誇り |

## 2. Lv1 → Lv5 の成長メソッド（普遍ルール）

**原則**: シルエットを変えず、"同じ家族" として育つ。

| レベル | サイズ | 変化の中心 | 装飾の濃度 | 周囲の要素 |
|---|---|---|---|---|
| Lv1 | 64×52 | 素朴な土台・1 階建て | 最小限（看板のみ） | なし |
| Lv2 | 64×72 | 2 階増築 or 庇追加 | 植木・小ベンチ追加 | ミニ植栽 |
| Lv3 | 64×92 | 側面に小部屋・ラテラル拡張 | 看板リニューアル・暖簾 | 花壇・自転車 |
| Lv4 | 64×112 | 塔 or 装飾屋根 | ライトアップ装飾 | テラス・小道 |
| Lv5 | 64×132 | ランドマーク化・屋上装飾 | 最大（旗・彫像・時計） | 噴水・彫刻・行列 |

### 成長カーブのメタファー

> **Lv1 = 若木 / Lv3 = 並木 / Lv5 = 大樹**

各レベルで周辺の「気配」が広がっていく。Lv5 では NPC が自然と建物の周りに集まる演出を追加。

## 4. プレミアム解放時の視覚変化

プレミアム購入で以下が有効化:

- ✨ **建物に金の縁取り**が追加（Lv5 のみ）
- ✨ **市庁舎に金の旗**がはためく
- ✨ **街全体の彩度 +10%**
- ✨ **夜の窓明かり色が暖色系に**（通常は青白）
- ✨ **道が石畳パターンに変化**（無料版は土）

---

## 5. 近接配置ボーナス（将来実装用設計）

> Phase 6 以降の拡張で、隣接配置により視覚的な "繋がり" を生む。

例:
- B001 ジム × B003 公園 が隣接 → 間に舗装された小道が自動描画
- B007 カフェ × B008 マーケット が隣接 → 露天のパラソルが自動追加
- B013 瞑想センター × B018 天文台 が隣接 → 月見台が自動生成

**設計ポリシー**: 配置ボーナスは "見た目だけ" の報酬。CP 計算への影響はしない（ゲームバランス保護）。

## 6. 制作優先順位（既存 Prompts と連携）

1. **MVP Top Priority**（`Prompts/01`） → B025 市庁舎 Lv1/3/5、B001/B003/B007 を先行
2. **High Priority**（`Prompts/02`） → 各軸の代表建物 Lv1/3 を制作
3. **Penalty**（`Prompts/03`） → B029/B030
4. **Remaining**（`Prompts/04`） → その他の Lv1
5. **All Lv5** → プレミアム訴求用に Lv5 を最後に一気に制作

## 7. 品質チェックリスト（各建物 完成判定）

- [ ] シルエットが同軸の他建物と区別できる
- [ ] Lv1 と Lv5 が "同じ家族" に見える（色相・窓形が揃っている）
- [ ] 1 つ以上の "物語装飾"（看板・暖簾・黒板・ベンチ等）がある
- [ ] 影が左上光源で統一されている
- [ ] 黒の輪郭線を使っていない（同系暗色のみ）
- [ ] 軸色が差し色として 1〜3 箇所に配置されている
- [ ] 透過 PNG で背景が完全透過
