# 02 — VITA CITY キャラクターデザイン設計書

> **対象**: NPC（街の住民）5 種 + 特殊キャラ 2 種
> **目的**: プレイヤーが街に親しみを感じる "住民" の人格と見た目を設計する。
> **前提**: `00_DesignOverview.md` の世界観「健康行動の具現化としての住民」を遵守。

---

## 1. 基本仕様（全キャラ共通）

| 項目 | 仕様 |
|---|---|
| サイズ | **24×42 px**（立ち絵。論理表示） |
| 描画 | PixelLab で 64×128 生成 → ダウンサンプル |
| 頭身 | **2.5 頭身のちびキャラ** |
| パーツ | 頭・胴・腕・脚の 4 パーツ（歩行アニメ 2 コマ） |
| 視点 | 正面 3/4 ビュー（アイソメトリック整合） |
| 輪郭 | 黒線ではなく同系色の暗色（肌: 茶、服: 色のシャドウ側） |
| シルエット | 肩幅 > 頭幅 にしない（= 頭大きめで親しみ） |
| 歩行速度 | 0.5 タイル/秒（CLAUDE.md 基調、焦らせない） |

### 1.1 歩行アニメーション規則

```
フレーム 1: 左足前、右腕前  (0.0s〜0.25s)
フレーム 2: 右足前、左腕前  (0.25s〜0.5s)
→ 繰り返し
```

- 腕振りは控えめ（± 2px）
- 上下バウンス ±1px で "歩いている感" を演出

### 1.2 スタンディング時の揺れ

- 立ち止まっている NPC も、**0.8 秒周期で ±1px の呼吸アニメ**
- "生きている感" を担保

## 2. NPC 5 種のキャラクター設計

### 2.1 住民 A — "朝ランナー ハルナ"（運動 × 若者）

| 項目 | 設計 |
|---|---|
| **人格** | 明るく元気。朝のランニングが日課。SNS 映えする写真を撮る |
| **物語** | 以前は運動嫌いだったが、旅人（プレイヤー）が建てた公園で変わった |
| **外見** | ショートカット・緑のトラックウェア・白いスニーカー |
| **配色** | 髪: 濃茶 / 服上: 緑 (#34C759) / 服下: 白 / 肌: やや日焼け |
| **持ち物** | 小さなウォーターボトル（腰に） |
| **固定ポーズ** | ストレッチ姿勢で立ち止まることあり |
| **出現条件** | 運動 CP 累計 500 以上で出現確率上昇 |
| **話すセリフ例** | 「今日も走ってきた！」「風が気持ちいい」「次はあの坂に挑戦」 |

### 2.2 住民 B — "カフェ店主 ミツル"（食事 × 中年）

| 項目 | 設計 |
|---|---|
| **人格** | 穏やかで職人気質。素材の話を語り出すと止まらない |
| **物語** | 元は都会の高級料理人。移住して B007 オーガニックカフェを開いた |
| **外見** | 白いコック帽 or バンダナ・オレンジのエプロン・チノパン・少し体格良し |
| **配色** | 髪: 白髪混じりの黒 / エプロン: 橙 (#FF9500) / ズボン: ベージュ |
| **持ち物** | お盆 or 野菜籠 |
| **固定ポーズ** | 腕組み（味見の顔）で立つことあり |
| **出現条件** | 食事 CP 累計 500 以上 |
| **話すセリフ例** | 「今朝のトマトは最高」「朝食は食べた？」「素材の力を信じて」 |

### 2.3 住民 C — "瞑想家 ユキ"（飲酒自制 × 静謐）

| 項目 | 設計 |
|---|---|
| **人格** | 静かで落ち着いている。寡黙だが芯が強い |
| **物語** | 過去に飲酒で体を壊し、瞑想と出会って生まれ変わった |
| **外見** | 紫の作務衣 or 和風ローブ・下駄 or 草履・長い黒髪を束ねる |
| **配色** | 髪: 黒 / 服: 紫 (#AF52DE) + 生成り / 帯: 深紫 |
| **持ち物** | 数珠 or 小さな香炉 |
| **固定ポーズ** | 合掌して目を閉じる（静止時） |
| **出現条件** | 飲酒軸 CP 累計 500 以上、または 3 日連続 "飲まない" |
| **話すセリフ例** | 「深呼吸しよう」「静けさに感謝」「月が綺麗だ」 |

### 2.4 住民 D — "夜更かし天文家 リク"（睡眠 × 知的）

| 項目 | 設計 |
|---|---|
| **人格** | マイペース、睡眠の専門家として街に助言する |
| **物語** | 睡眠障害を克服した経験から、B017 睡眠クリニックを支える |
| **外見** | 青いナイトキャップ or ベレー帽・白衣 or 青いパジャマ風・丸眼鏡 |
| **配色** | 髪: 茶 / 帽子: 青 (#007AFF) / 服: 白 + 青 |
| **持ち物** | 星座図 or 小さな望遠鏡 |
| **固定ポーズ** | 空を見上げる |
| **出現条件** | 睡眠 CP 累計 500 以上 |
| **話すセリフ例** | 「よく寝れた？」「今夜は月が綺麗」「7 時間が黄金律」 |

### 2.5 住民 E — "司書 アオイ"（生活習慣 × 温和）

| 項目 | 設計 |
|---|---|
| **人格** | 優しい姉御肌。すべての住民の相談相手 |
| **物語** | B019 図書館の司書。本の知識で街の健康を支える |
| **外見** | ピンクのカーディガン・長めのスカート・三つ編み or まとめ髪・丸眼鏡 |
| **配色** | 髪: 栗色 / カーディガン: ピンク (#FF2D55, 淡め) / スカート: 濃紫 |
| **持ち物** | 本 2〜3 冊 or ティーカップ |
| **固定ポーズ** | 本を読みながら歩く |
| **出現条件** | 生活習慣 CP 累計 500 以上、または 7 日連続ストリーク |
| **話すセリフ例** | 「いい本があるよ」「水、飲んだ？」「継続は力なり」 |

## 3. 特殊キャラクター 2 種

### 3.1 アドベンチャラー "旅人"（プレイヤーの化身）

| 項目 | 設計 |
|---|---|
| **役割** | プレイヤー自身の象徴。市庁舎付近に常駐 |
| **出現条件** | 街レベル 3 以上で登場 |
| **外見** | 金のマント（プレミアム時）/ 茶の旅人服・大きめのリュック・帽子 |
| **配色** | 髪: 金 / 服: 茶 + 金の縁取り / マント: 金 (#FFD700) |
| **持ち物** | 地図 or 杖 |
| **固定ポーズ** | 腕組みで街を見下ろす |
| **話すセリフ例** | 「ここまでよく頑張ったね」「街が育ってきた」「次の旅路を考えよう」 |

### 3.2 郵便配達員 "ポッピー"（ログインボーナス届け人）

| 項目 | 設計 |
|---|---|
| **役割** | 日替わりログインボーナス受取オーバーレイに登場するマスコット |
| **外見** | ピンクの制服・小さな帽子・肩から斜めがけのメールバッグ |

## 5. NPC の出現・密度ロジック

実装は `CitySceneCoordinator.updateStepCount(_:)` で制御済み。設計上の方針:

### 5.1 出現数の計算

| 歩数 | 表示 NPC 数 | 補足 |
|---|---|---|
| 0〜999 歩 | 0 | 街が無人で寂しい（行動を促す） |
| 1,000 歩 | 2 | NPC が出始める |
| 5,000 歩 | 10 | 街に生活感 |
| 10,000 歩 | 20 | 賑わい |
| 15,000 歩以上 | 50（上限） | お祭り状態 |

### 5.2 ロール比率

歩数に応じて住民 A〜E が増えていくが、**偏らないように全 5 種から均等に出現**させる。
CP 累計条件を満たしたロールが優先的に増える。

```
早期（歩数 1000〜5000）: Citizen1（ハルナ）+ Citizen2（ミツル）中心
中期（5000〜10000）: 全 5 種が 2 人ずつ混ざる
後期（10000+）: 全 5 種 + 旅人 1 人 + ポッピー 1 人
```

### 5.3 位置ロジック

- 各 NPC は **所属する軸の建物近辺** を好んで歩く
  - 例: ハルナは公園・ジム・スタジアム周辺
  - アオイは図書館周辺
- 建物 Lv5 の周囲には常に 1〜2 人の NPC が立ち止まる（賑わい演出）

## 6. NPC インタラクション設計（実装済み）

実装は `NPCNode.touchesBegan`。設計上の意図:

| 動作 | 仕様 |
|---|---|
| タップ応答 | **必ず応答する**（無視しない = 孤独感を避ける） |
| 吹き出し表示時間 | 2.5 秒（長すぎず短すぎず） |
| 吹き出し色 | 白背景 + ロール軸色の縁取り（将来実装） |
| リアクションアニメ | 軽く飛び跳ねる（± 3px、0.25s） |
| 触覚フィードバック | `HapticEngine.tapLight()` |

## 7. 感情表現の記号（吹き出し内アイコン）

吹き出しに小さなアイコンを添えると親しみが増す:

| 感情 | アイコン | 使用シーン |
|---|---|---|
| 嬉しい | `❤️` / ハート | ログインボーナス、ストリーク達成時 |
| 応援 | `✊` / 拳 | タップ時のランダム |
| 眠い | `💤` / zzz | 夜間時間帯 |
| 驚き | `!` | レベルアップ時 |
| 感謝 | `🙏` | 建物建設完了時 |

実装方針: 絵文字は使わず、**ドット絵の小アイコン 16×16 px** を用意。

## 10. アクセシビリティ配慮

- シルエットの輪郭がはっきり → 色覚異常でも識別可能
- ロールごとに **服の形状が異なる**（色だけに頼らない）
  - ハルナ: スリム、スニーカー
  - ミツル: エプロンで胴体が隠れる
  - ユキ: 和装
  - リク: 帽子 + 白衣
  - アオイ: ロングスカート
- タップ判定領域は **sprite サイズ + 8px マージン**（ボタン押しやすさ）

## 11. 品質チェックリスト

- [ ] 2 コマの歩行アニメが自然
- [ ] 呼吸アニメ（待機時 ± 1px）が入っている
- [ ] 軸色が服の 1 箇所以上に使われている
- [ ] 他ロールと一目で区別できる（形状 + 色）
- [ ] 建物の前に立った時に隠れすぎない（NPC サイズ 24×42 厳守）
- [ ] 左上光源の影が右下に落ちている
